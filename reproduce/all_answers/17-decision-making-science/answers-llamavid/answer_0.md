The Decision Lens software improves upon traditional AHP methods in several ways:

1. It allows for continuous planning across time horizons, rather than just a single model. This allows for more dynamic and real-time decision making. 

2. It connects across functions, rather than just silos. This allows for better integration of strategic objectives across departments.

3. It allows for more efficient trade-offs across different resources, rather than just a single model. This allows for more optimal allocation of resources.

4. It allows for more advanced optimization and prioritization of investments, based on a larger set of criteria. This allows for more effective portfolio planning.

5. It allows for more real-time monitoring and adjustment of priorities, based on changing conditions. This allows for more agile and adaptive decision making.

In summary, the Decision Lens software allows for more dynamic, real-time, and adaptive decision making, based on a larger set of criteria and across multiple functions and resources. This improves upon traditional AHP methods, which are more static and siloed.
