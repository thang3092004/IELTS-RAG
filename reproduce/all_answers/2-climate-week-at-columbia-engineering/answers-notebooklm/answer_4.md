Symbolic regression can be used to discover equations based on data, which can then be used to improve climate modeling.

One way this is done is by exposing a large dataset to equations. This process allows symbolic regression to learn an equation from the data. As an example, an equation for cloud cover can be learned based on temperature and moisture data. Equations are easier for scientists to interpret compared to neural networks. This increased interpretability builds trust in the climate community.

Symbolic regression can also produce results that are very close to observations. For example, when using symbolic regression, cloud cover predictions are very similar to high resolution products and observations.
