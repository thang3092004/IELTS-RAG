## Calculating Global Priorities in AHP Using Super Decisions

The Analytic Hierarchy Process (AHP) within the Super Decisions software offers a structured and systematic approach for calculating global priorities of alternatives. This method allows decision-makers to evaluate and rank alternatives based on a defined set of criteria.

### Structured Decision Hierarchy

To begin, users will need to establish a decision hierarchy. This hierarchy typically starts with the main goal at the top, followed by various criteria and sub-criteria that are pertinent to the decision at hand. Each alternative is then assessed against these criteria, enabling a thorough comparison across the board.

### Pairwise Comparisons

Super Decisions facilitates the process of pairwise comparisons among alternatives for each criterion. Users input their judgments through a Comparison Matrix, where each element represents a comparison between pairs of criteria or alternatives. This scoring system generates numerical values that reflect the relative importance of each alternative.

### Calculation of Local and Global Priorities

Once the pairwise comparisons are completed, the software calculates local priority scores for each alternative based on the individual criterion assessments. These local scores are then aggregated throughout the hierarchy to compute global priorities. This step involves multiplying local priorities by the corresponding weights assigned to each criterion, ensuring that decisions are data-driven and reflective of user assessments. The final outcome is a set of global priority scores that indicates how each alternative ranks overall.

### Consistency and Sensitivity Analysis

Additionally, Super Decisions includes tools for evaluating the consistency of the judgments made during pairwise comparisons. A low Consistency Ratio signifies high reliability in the judgments, essential for credible priority calculations. Sensitivity analysis features allow users to observe how changes in priorities impact global outcomes, providing insight into the robustness of the decisions made.

### Conclusion

In summary, the AHP within Super Decisions employs a structured framework for users to compute global priorities by utilizing pairwise comparisons and systematically aggregating scores. This method not only ensures thorough evaluation but also enhances transparency and robustness in decision-making processes.
