The two phases involved in implementing absolute measurement in SuperDecisions software are:

1. Developing the model using relative measurement. This involves creating clusters for the criteria and alternatives, then pairwise comparing the criteria using the SuperDecisions software.

2. Converting the model to absolute measurement using the ratings model. This involves defining rating scales for the criteria and alternatives, then entering judgments for the criteria and alternatives. The priorities are calculated by the SuperDecisions software using the weights from the model.
