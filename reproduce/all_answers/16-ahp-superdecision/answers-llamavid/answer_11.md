The ratings mode in SuperDecisions software provides several advantages for absolute measurement. First, it allows for the evaluation of a large number of alternatives in a multi-level network. This is useful for situations where there are many options to consider, such as selecting the best company for a job. Second, it allows for the use of existing rating scales that have already been developed for each criterion, saving time and effort in creating new scales. Third, the ratings mode allows for the use of direct judgments that are already known, rather than having to perform pairwise comparisons between alternatives. This speeds up the evaluation process. Finally, the ratings mode provides a synthesis of the priorities for each alternative, which can be used for sensitivity analysis to identify the most important criteria and sub-criteria. Overall, the ratings mode provides a streamlined and efficient way to evaluate alternatives in a multi-level network using SuperDecisions software.
