The 'inconsistency report' feature in SuperDecisions can be used to identify and rectify inconsistencies in pairwise comparison judgments. This report shows the inconsistency ratio between the priorities of the criteria and the alternatives, which is calculated by dividing the difference between the priorities of the criteria and alternatives by the sum of the weights of the criteria. If the ratio exceeds 0.01, it indicates an inconsistency. The report also shows the inconsistency ratio for each alternative and the criteria, which can be used to identify which criteria and criteria are causing the inconsistency. This report can be used to identify which criteria and criteria need to be revisited to improve consistency.
