Rating scales are an important tool when creating a ratings model in SuperDecisions. They allow you to assign numerical values to each alternative based on how important a criterion is for that alternative. However, defining and utilizing rating scales can be challenging. Here are some tips for defining and utilizing rating scales in a ratings model:

1. Define your rating scales: Define your rating scales based on the criteria and alternatives. For example, assign a numerical value of 1 to 10 to each alternative for each criterion.

2. Use existing scales: Use existing scales where possible. You can save your rating scales in files with the extension "rcp". This makes it easier to use the same scales in multiple models.

3. Assign numerical values: Assign numerical values to each alternative based on how important the criteria are for that alternative. The values should add up to 10.

4. Perform sensitivity analysis: Perform sensitivity analysis to see how the ratings change when you change the weights of the criteria. This can help you identify which criteria are most important.

5. Perform pairwise comparisons: Perform pairwise comparisons between alternatives to rate the alternatives based on the criteria.
