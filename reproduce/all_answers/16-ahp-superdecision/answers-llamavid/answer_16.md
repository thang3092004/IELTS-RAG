In Analytic Hierarchy Process (AHP) modeling, a key tool in decision making, is used to obtain global priorities for alternatives. AHP models have a goal node, criteria nodes representing criteria, and alternatives nodes representing alternatives. 


To synthesize an AHP model in SuperDecisions, first create a network model using nodes representing the goal, criteria, and alternatives. Then connect the goal to the criteria nodes. Next, create a ratings model using judgments for the criteria and alternatives. Finally, synthesize the model to obtain global priorities for alternatives by calculating the weights of the criteria and alternatives nodes.



In summary, in SuperDecisions, AHP modeling is used to synthesize global priorities for alternatives by connecting the goal node to the criteria nodes, rating the criteria nodes, and synthesizing the model to obtain the global priorities.
