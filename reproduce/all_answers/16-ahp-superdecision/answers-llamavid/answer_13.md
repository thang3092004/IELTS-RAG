To establish connections between different elements in an AHP hierarchy within SuperDecisions, you can follow these steps:

1. Create a new cluster for the goal node.
2. Connect the goal node to the criteria node.
3. Connect the criteria node to the alternatives node.
4. Connect the criteria node to the criteria node.
5. Connect the criteria node to the sub criteria node.

6. Connect the sub criteria node to the alternatives node.

7. Connect the alternatives node to the alternatives node.

8. Check the connections between the nodes to ensure they are correctly connected.

9. Perform pairwise comparisons between nodes to form the pairwise comparison matrices.

10. Synthesize the results and compute the priorities.

11. Perform sensitivity analysis to analyze the impact of changing the weights of different criteria.

12. Perform a dynamic sensitivity analysis to see how the priorities change with different weights.
