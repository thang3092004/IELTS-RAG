Creating an AHP hierarchy in SuperDecisions software involves defining the goal, criteria, and alternatives. The first step is to define the goal node, which represents the ultimate goal or decision to be made. Next, the criteria nodes are created by connecting the goal to the criteria nodes. Finally, the alternatives nodes are created by connecting the criteria nodes to the alternatives nodes. The goal, criteria, and alternatives nodes are then connected to form the AHP hierarchy. Once the AHP hierarchy is defined, the priorities can be computed using SuperDecisions software.
