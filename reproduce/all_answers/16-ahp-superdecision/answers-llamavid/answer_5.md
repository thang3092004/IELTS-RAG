The inconsistency ratio can be improved by using the Constant Ratio method, which involves rearranging the nodes to create a consistent ratio between the criteria weights and alternative weights. The method works by rearranging the nodes in the supermatrix so that the weights of the criteria and alternatives are in a constant ratio. For example, if the weights of the criteria are 2, 3, 4, then the weights of the alternatives are 1, 1.5, 2.5, 3.5, 4, etc. The method can be applied to the pairwise comparison matrices and supermatrix weights in Super Decisions software to improve consistency ratios.
