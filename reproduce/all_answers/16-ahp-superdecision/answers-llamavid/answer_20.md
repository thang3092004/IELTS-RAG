When creating a SuperDecisions model, it is important to properly connect criteria and sub-criteria to alternatives in order to properly link the criteria and sub-criteria to the alternatives in the hierarchy. The steps involved in connecting criteria and sub-criteria to alternatives in SuperDecisions are:

1. Create a new node for the goal, for example "best company"
2. Connect the goal node to the criteria nodes, for example, "salary", "research", "working environment", etc.
3. Connect the goal node to the alternatives nodes, for example, "Company A", "Company B", etc.
4. Check that the connections between nodes are correct and that no nodes are left unconnected.
5. Check the pairwise comparison matrices for the criteria and alternatives.
6. Perform sensitivity analysis to analyze how the weights of the criteria and alternatives change with different values of the criteria.

By following these steps, you can properly link the criteria and sub-criteria to the alternatives in the SuperDecisions model hierarchy.
