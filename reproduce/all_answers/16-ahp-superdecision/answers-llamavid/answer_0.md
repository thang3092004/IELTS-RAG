A ratings model is useful when there are many alternatives to evaluate, and it can help overcome the limitations of pairwise comparisons. The main benefits of a ratings model are:

1. It can handle a large number of alternatives, which is particularly useful when evaluating a large number of projects. 

2. It can provide a relative ranking of alternatives, rather than an absolute ranking. This allows for comparing alternatives that are not directly comparable.

3. It can assign a priority weight to each alternative, which can help in making decisions.

4. It can synthesize the priorities from the judgments, which can be used to perform sensitivity analysis.

In summary, a ratings model provides a more flexible and powerful way to evaluate alternatives in AHP, especially when there are many options to consider. It allows for a relative ranking of alternatives, rather than an absolute ranking, and can provide a priority weight that can help make decisions.
