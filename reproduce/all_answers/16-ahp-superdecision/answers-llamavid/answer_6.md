The AHP model is a multi-level decision making tool that allows for decision making at different levels. Sub-criteria can be incorporated into the AHP hierarchy using the ratings model in Super Decisions. 


To do this, first create a new cluster for the sub-criteria. Then add the sub-criteria nodes to the clusters. Next, enter judgments for each sub-criteria node. Finally, use the pairwise comparison mode to compare the nodes. The priorities can then be calculated using the super decision software.


This allows the sub-criteria to be incorporated into the AHP hierarchy at the same level as the main criteria, providing a more robust decision making model.
