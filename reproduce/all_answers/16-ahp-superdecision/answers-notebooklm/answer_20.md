The sources detail the process of connecting criteria and sub-criteria to alternatives in SuperDecisions to establish a hierarchical decision-making model. Here are the key steps:

1.  **Establish Clusters:** Begin by creating clusters in SuperDecisions. These clusters represent different levels of your hierarchy, typically including Goal, Criteria, and Alternatives. If a criterion has sub-criteria, create additional clusters to represent those as well. 
2. **Create Nodes:** Within each cluster, create nodes representing the specific elements at that level.  For instance, under the "Criteria" cluster, you would create nodes for each criterion, such as "Salary," "Research Growth," and "Working Environment." Under the "Alternatives" cluster, you would list the options being considered, such as "Company A," "Company B," and "Company C." If dealing with sub-criteria, create nodes for them under their corresponding sub-criteria cluster.
3. **Connect Goal to Criteria:** Select the "Goal" node and utilize the "Node Connection Form" to connect it to all the criteria nodes. This visually establishes the relationship between the goal and the factors influencing the decision.
4. **Connect Criteria to Alternatives:** For criteria *without* sub-criteria, select each criterion node and, using the "Node Connection Form," connect it to all the alternative nodes. This illustrates how each criterion is evaluated against the different alternatives.
5. **Disconnect Criteria with Sub-Criteria:** If a criterion has sub-criteria, you must first disconnect it from the alternatives. Right-click on the criterion node, select "Node Connection Form," and deselect the alternatives.
6. **Connect Sub-Criteria to Parent Criterion:** Select the parent criterion node and use the "Node Connection Form" to connect it to its sub-criteria nodes. For example, if "Research" has sub-criteria like "Grant" and "Library Facility," connect the "Research" node to both "Grant" and "Library Facility."
7. **Connect Sub-Criteria to Alternatives:** Finally, select each sub-criterion node and utilize the "Node Connection Form" to connect it to all the alternative nodes. This completes the hierarchical linkage, demonstrating how the sub-criteria are assessed against each alternative.

By following these steps, you successfully connect criteria and sub-criteria to alternatives within the SuperDecisions framework. This structured connection process ensures a clear representation of the hierarchical relationships between the elements of your decision model, facilitating a comprehensive analysis and evaluation process.
