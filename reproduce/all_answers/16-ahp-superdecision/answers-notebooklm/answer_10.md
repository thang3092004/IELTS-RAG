The absolute measurement process in SuperDecisions software is used when there are a large number of alternatives. It is implemented in two phases:

**Phase 1:** The model is developed using relative measurement. This is the same process as the relative measurement process.

**Phase 2:** The ratings mode is used. In this phase:

* Ratings are inserted.
* Ratings are prioritized.
* The performance of the candidates is provided.
* The software computes the weightages.
