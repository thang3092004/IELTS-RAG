To establish connections between various elements in an AHP (Analytic Hierarchy Process) hierarchy within the SuperDecisions software, you can follow these steps:

**1. Create Clusters:** Begin by creating clusters that represent the different levels of your hierarchy. These clusters serve as containers for the nodes or elements within each level. For instance, you might have clusters for "Goal," "Criteria," "Sub-criteria," and "Alternatives."

**2. Insert Nodes:** Within each cluster, insert nodes to represent the specific elements at that level. For example, in the "Criteria" cluster, you would create nodes for each criterion you are considering.

**3. Establish Connections:** Once the nodes are in place, you can establish connections between them to represent the relationships within the hierarchy. To do this, select the parent node (e.g., "Goal") and use the "Design Node Connection Form" to choose the child nodes (e.g., criteria) that should be connected to it.

**4. Verify Connections:** After making the connections, use the "Connection Mode" to visually verify that the nodes are correctly linked. In this mode, selecting a node will highlight all its connected nodes with a red border.

**5. Connect Alternatives:**  If you have sub-criteria, ensure that you connect the sub-criteria nodes to the relevant alternatives, not the main criteria node. You may need to deselect the connection between the main criterion and the alternatives before establishing connections between the sub-criteria and alternatives.

Following these steps will help you establish the necessary connections between the elements in your AHP hierarchy within the SuperDecisions software, enabling you to proceed with pairwise comparisons and ultimately synthesize the model to obtain overall priorities.
