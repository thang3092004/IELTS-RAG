When there is a large number of alternatives in the Analytic Hierarchy Process (AHP), **absolute measurement is preferred over relative measurement because relative measurement becomes less suitable**. Absolute measurement is implemented in two phases:

*   The first phase involves developing the model using relative measurement, the same process used in traditional AHP. 
*   The second phase utilizes a ratings mode.

Using absolute measurement with a ratings mode in AHP is more practical with a large number of alternatives because pairwise comparisons would be too time-consuming.
