The sources indicate there are a couple of key benefits to using a ratings model, as opposed to pairwise comparisons, when working with the Analytic Hierarchy Process (AHP). The most important benefit is that **ratings models are much faster to use than pairwise comparisons**, especially when there are many alternatives to consider. However, **it is important to note that ratings models require a higher degree of expertise in the subject matter** to be effective. Since there are no pairwise comparisons in a ratings model, the user needs to be able to confidently evaluate each alternative against an ideal in their mind without comparing the alternatives to each other.

The sources provide a specific use case where ratings models may be particularly helpful. **When there are too many alternatives to reasonably perform pairwise comparisons, a ratings model is a good alternative.** Because ratings models can be created much faster than pairwise comparison models, this can save a lot of time for the user.

However, the sources make it clear that if the user begins working with a ratings model and finds they do not have the subject matter expertise to properly evaluate the alternatives, **it is possible to revert a ratings model back to a pairwise comparisons model.**
