In SuperDecisions software, performing sensitivity analysis for an AHP model involves the following steps:

*   **Open an existing AHP model in SuperDecisions.**
*   **Access the sensitivity analysis feature**. In SuperDecisions, this is done by going to "Computations," then selecting "Influence/Sensitivity," then "Options," then "Smart," then "p0," and finally "Computations" and "Node Sensitivity."
*   **Choose the criterion to analyze.** This can be done through a dropdown menu or by selecting the desired criterion on the sensitivity analysis plot.
*   **View and interpret the sensitivity results.** The software presents the results in various formats, including:
    *   **Sensitivity plot:** Displays how the priorities of the alternatives change as the weight of the chosen criterion varies. This plot often highlights "rank reversals," points where the preferred alternative changes due to the criterion's weight exceeding a certain threshold.
    *   **Bar chart:** Shows the changing priorities of the alternatives as a bar graph. This visualization can be adjusted by changing the parameter (weight) of the selected criterion.
    *   **Pie chart:** Provides another visual representation of the alternatives' priorities.
    *   **Horizontal bar chart:** Presents the rankings of the alternatives based on their priorities.
*   **Dynamic sensitivity analysis:** This feature, accessed through "Computations," "Node Sensitivity," and "Graphical Sensitivity," allows users to interactively explore how changes in a criterion's weight affect the alternatives' values. By adjusting the parameter "P," which represents the weight of the chosen criterion, users can observe how the alternative values dynamically change, providing insights into the model's sensitivity.

The type of sensitivity analysis used depends on the purpose. **Single sensitivity analysis is helpful for understanding how alternative solutions are impacted by changes in the weight of a specific criterion**. **Dynamic sensitivity analysis offers a broader view of how alternative values change when a specific criterion is varied.**
