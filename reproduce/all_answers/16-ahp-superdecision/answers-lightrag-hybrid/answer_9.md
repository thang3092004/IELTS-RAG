## Performing Sensitivity Analysis in SuperDecisions for an AHP Model

Sensitivity analysis within the SuperDecisions software involves a systematic process aimed at evaluating how changes in the criteria weights affect the preferences among different alternatives. The following steps outline the essential procedures involved in conducting sensitivity analysis for an Analytic Hierarchy Process (AHP) model:

### Step 1: Setup the AHP Model
Begin by creating a decision-making model using the AHP framework within SuperDecisions. This includes defining the goal node, criteria, and alternatives to be evaluated. Enter relevant data and establish the relationships between nodes that represent the decision criteria and the options available for evaluation.

### Step 2: Input the Pairwise Comparisons
Once the model is set up, the next step is to conduct pairwise comparisons. This involves the user evaluating the relative importance of each criterion against others. For instance, if you are comparing criteria such as price, comfort, and mileage, you would pair each criterion and input qualitative or quantitative judgments that reflect their importance relative to one another.

### Step 3: Activate Sensitivity Analysis Features
After establishing your pairwise comparison matrix, navigate to the analysis features of SuperDecisions. The software provides different methods for sensitivity analysis. Users can opt for **simple sensitivity analysis**, which examines the robustness of the model by altering the weights of the criteria one at a time, or **dynamic sensitivity analysis**, which shows how priorities shift among the alternatives as multiple criteria weights are modified simultaneously.

### Step 4: Conduct Sensitivity Analysis
Use the analysis tools in SuperDecisions to run the sensitivity analysis. Users can adjust the weight of each criterion to observe its effect on the overall prioritization of alternatives. For example, increasing the weight of the comfort criterion may show changes in rankings among the vehicles being assessed, like how it influences the preference for the Honda Civic compared to others.

### Step 5: Analyze Results
Evaluate the output from the sensitivity analysis. Visual aids, such as graphs and charts, may display how changes in criteria weights affect alternative rankings. SuperDecisions enables users to view these results in multiple formats, including bar charts and pie charts, to better understand the implications of different criteria settings on decision outcomes.

### Step 6: Document Insights
Make notes of critical insights derived from the sensitivity analysis. This may include points of rank reversal (where the order of preferences changes based on different weight scenarios) and the robustness of the selections based on various criteria adjustments. These findings can support decision-making processes or help refine the models for better accuracy.

### Conclusion
By following these steps in SuperDecisions, users can effectively perform sensitivity analysis for AHP models. This process not only facilitates better understanding of how different criteria impact decisions but also aids in making informed choices based on dynamic evaluations of alternative options. Being aware of these methods enhances the quality of analyses conducted within the software, leading to improved decision-making outcomes in various contexts, including vehicle selection, project prioritization, and more.
