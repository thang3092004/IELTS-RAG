### Visualization Options in SuperDecisions for Sensitivity Analysis Results

SuperDecisions provides a variety of visualization options to effectively display and analyze sensitivity analysis results. These visual tools enhance the user experience and help in better decision-making by enabling users to interpret data more intuitively. Here are the key visualization options available:

#### 1. **Bar Charts**
Bar charts are widely used in SuperDecisions to represent the performance data of different alternatives visually. This format allows users to compare the effectiveness of different criteria, making it easier to understand how each alternative ranks based on user-defined metrics such as price, prestige, and comfort. Throughout the sensitivity analysis process, users can adjust parameters and observe how these changes impact the bar chart's depiction of outcomes.

#### 2. **3D Bar Charts**
For a more dynamic representation, SuperDecisions includes a specialized 3D bar chart that displays multiple data points related to alternatives. This visualization can effectively communicate comparative information, allowing users to grasp relationships and differences among various options more visually and distinctly.

#### 3. **Pie Charts**
Pie charts serve as an option for visualizing data distribution across alternatives. They provide a straightforward way to present proportions and comparisons, helping users quickly identify which options stand out under specific analytical conditions.

#### 4. **Scatter Plots**
Scatter plots are another option within the SuperDecisions interface to visualize results based on various parameters. This format can be particularly useful for determining correlations and relationships between multiple criteria, thereby aiding users in understanding the dynamics of their decision-making framework.

#### 5. **Computational Sensitivity Graph**
The Computational Sensitivity Graph is specifically designed to visualize the results of sensitivity analysis. It represents how different variable adjustments impact decision choices, enabling users to interpret complex data relationships effectively.

#### 6. **Matrix Visualization**
Using a matrix format, SuperDecisions provides a layout designed to facilitate pairwise comparisons among criteria or alternatives. This allows for a structured view of how each option measures up across various dimensions and can highlight inconsistencies in the evaluations.

### Conclusion

The diverse visualization tools available in SuperDecisions significantly contribute to the clarity and effectiveness of sensitivity analysis, ensuring that users can not only analyze data but also communicate their findings efficiently.
