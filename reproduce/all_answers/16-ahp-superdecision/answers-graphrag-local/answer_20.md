In SuperDecisions, connecting criteria and sub-criteria to alternatives is a structured process that ensures a proper hierarchy is established for effective decision-making. This involves several key steps that facilitate the integration of various decision elements within the software. Below is a detailed overview of these steps:

### Step 1: Define the Criteria and Sub-Criteria
The first step is to establish the criteria and sub-criteria that will be used to evaluate the alternatives. This involves identifying the primary metrics that are essential for the decision-making process. For example, criteria may include aspects such as price, comfort, and fuel efficiency, with sub-criteria further detailing specific evaluation points like maintenance cost or seating capacity.

### Step 2: Set Up the Hierarchy
After defining the criteria, the next step is to organize these elements into a hierarchical structure. In SuperDecisions, this is typically achieved using the Criteria Node feature, where users can visually arrange the criteria and sub-criteria. The hierarchy allows users to see how criteria are related to one another, with main criteria on the upper levels and sub-criteria branching below.

### Step 3: Create Alternatives
Once the criteria hierarchy is set, the next step is to identify the alternatives that will be evaluated. These alternatives represent the options available for selection based on the established criteria. For instance, if evaluating vehicles, the alternatives might include the Toyota Camry, Honda Civic, and Acura TL.

### Step 4: Establish Connections
With both the criteria and alternatives defined, the user can now create connections between them. This is done within the software's interface where users can draw links between specific criteria and the alternatives they pertain to. This step is crucial as it ensures that each alternative is evaluated against the relevant criteria and sub-criteria.

### Step 5: Input Judgments and Data
Following the connections, users need to enter judgments regarding the performance of each alternative against the defined criteria. This often includes subjective evaluations, such as rating comfort on a scale, which can then feed into the overall analysis.

### Step 6: Perform Sensitivity Analysis
Once the connections and judgments are established, sensitivity analysis can be employed to observe how changes in criteria weight impact the ranking of alternatives. Sensitivity analysis is crucial for understanding the robustness of the decisions made and identifying which criteria are most influential.

### Step 7: Synthesize Results
After analyzing the data, the final step is to synthesize the results. This involves generating overall priorities for the alternatives, providing a clear overview of which option best meets the established criteria. The synthesis consolidates the decisions into actionable insights, allowing users to make informed choices based on the analysis.

### Conclusion
Through these steps in SuperDecisions, users can create a well-structured decision-making model that effectively links criteria and sub-criteria to alternatives. This hierarchical approach not only facilitates clearer evaluations but also enhances the overall decision-making process by allowing for comprehensive sensitivity analyses and prioritization of options. By following these guidelines, users ensure a systematic approach to decision-making that leverages the full capabilities of the SuperDecisions software.
