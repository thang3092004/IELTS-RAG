To implement absolute measurement using SuperDecisions software within the Analytic Hierarchy Process (AHP) framework, two distinct phases are followed. Each phase is critical to ensure that decisions made using this method are both structured and informed.

### Phase 1: Definition and Setup

The implementation begins with early preparations where the foundational elements of the decision-making model are defined. In this phase, the user is required to set up the criteria, alternatives, and the overall structure of the decision-making problem. This involves:

- **Identifying Criteria:** The user must determine which criteria will be evaluated. For example, this may include performance metrics relevant to the decision context, such as grades, experience, or other relevant factors. In the context of an educational setting, parameters might include Admission Test Results (ATR), Cumulative Grade Point Average (CGPA), Work Experience (WE), Letters of Recommendation (LOR), and Extra-curricular Activities (EA).
  
- **Listing Alternatives:** Alternatives need to be defined clearly. In the example of selecting the best candidates for an MBA program, these could be individual applicants scored based on their identified criteria.

- **Software Navigation:** This phase heavily involves utilizing the SuperDecisions software interface to create a structured decision-making model. This requires understanding the software's functionalities, which have been demonstrated in the associated video tutorials. Actions include creating clusters for criteria and alternatives and establishing connections between them to depict the decision hierarchy accurately.

### Phase 2: Implementation of Absolute Measurement

After the structure and criteria are established, the focus shifts to the practical application of absolute measurement in SuperDecisions. This phase includes:

- **Conducting Pairwise Comparisons:** Users are instructed to use pairwise comparisons to evaluate the importance of each criterion in relation to the alternatives, applying numerical scales as needed. The operations performed here involve assessing each alternative against every criterion, determining their priorities based on predefined measurements.

- **Use of Numeric Values in Ratings:** In this step, absolute values are assigned to each of the alternatives concerning the developed criteria. Different options within the software allow users to input ratings directly or utilize a verbal or graphical method for entering these judgments, which will ultimately calculate priority vectors.

- **Normalization and Results Synthesis:** After ratings have been inputted, the software helps in synthesizing the results, normalizing the data to reflect fair comparisons among all alternatives. The synthesis process provides the user with a clear output indicating the priorities of each alternative, based on the absolute measurements undertaken.

### Conclusion

In summary, the two phases—definition and setup, followed by the actual implementation of absolute measurements—collaboratively facilitate a structured approach to decision-making within the SuperDecisions software environment. This dual-phase methodology ensures that not only are the criteria and alternatives thoughtfully considered, but the efficacy of the decisions made is also significantly enhanced through rigorous comparisons and synthesis of results.
