To effectively connect criteria and sub-criteria to alternatives in SuperDecisions, it’s essential to follow a systematic approach that adheres to the framework of the Analytic Hierarchy Process (AHP). Here are the key steps involved in ensuring proper linkage in the hierarchy:

### 1. **Open and Set Up the Model**
   Begin by launching the SuperDecisions software and opening the specific model you wish to work on or create a new one. When starting a new model, ensure you save it with an appropriate name related to your analysis.

### 2. **Create Clusters for Criteria and Alternatives**
   Once in the main window, you will need to create relevant clusters:
   - Go to the “Design” tab and select “Cluster” to add new clusters.
   - Create a cluster for **Criteria** and another for **Alternatives**. Label them accordingly, as these clusters will serve as containers for your nodes.

### 3. **Add Nodes to Each Cluster**
   After establishing the necessary clusters:
   - Click on the created **Criteria** cluster and add nodes that represent each criterion relevant to your decision-making. For example, nodes could include “Prestige,” “Price,” and “Comfort.”
   - Similarly, within the **Alternatives** cluster, add nodes that correspond to the options you are evaluating, such as “Acura TL,” “Toyota Camry,” and “Honda Civic.”

### 4. **Connect Criteria to Alternatives**
   After populating the clusters with nodes, the next step is to ensure that the criteria are connected to the alternatives:
   - Select a criterion node (e.g., “Prestige”). 
   - Use the “Node Connections” tool from the design menu, typically found under options like “Node Connections from” or “Node Connection Form.” This allows you to create connections from your selected criterion to each of the alternative nodes by selecting each alternative in a streamlined process.
   - Make sure that all relevant alternatives are connected to the criteria by repeating this process for each criterion.

### 5. **Verify Connections**
   Once all connections have been made:
   - Review the model visually to ensure each criterion node is directly linked to the appropriate alternative nodes. 
   - Utilize the connection mode or the “Show Connections” feature to visualize all existing links to correct any errors or missing connections.

### 6. **Enter Judgments for Criteria**
   With the linkages complete, proceed to the judgments portion:
   - Go to the assessment panel and enter pairwise comparisons for the criteria against each other, and according to each alternative. 
   - This step involves evaluating how important each criterion is for each alternative in relative terms.

### 7. **Check for Inconsistencies**
   As you enter judgments, it’s crucial to check for inconsistencies:
   - Utilize the inconsistency check feature within the software to ensure that the judgments made adhere to logical consistency thresholds.

### 8. **Finalize and Synthesize Results**
   After all connections and judgments are entered and validated:
   - Synthesize the results to derive overall priorities for both alternatives and criteria. This can usually be done through a computation tab where “Synthesizing” options are available.

### Conclusion
These steps outline a structured way to connect criteria and sub-criteria with alternatives within the SuperDecisions software. By systematically creating clusters, adding nodes, ensuring correct connections, and finalizing the judgments, users can develop a robust decision-making framework that accurately reflects their evaluation of alternatives based on identified criteria.
