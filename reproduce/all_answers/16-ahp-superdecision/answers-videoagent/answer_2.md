In Super Decisions, the results of AHP analysis can be visualized using various tools and features within the software. To visualize the priorities of alternatives and criteria, you can use the following methods:


1. **Hierarchy View**: This view allows you to see the entire hierarchy of your decision model, including criteria and alternatives. You can view the priorities directly on the hierarchy.


2. **Priorities Window**: This window displays the calculated priorities for each criterion and alternative. You can see the global and local priorities, which help in understanding the relative importance of each element.


3. **Sensitivity Analysis**: This feature allows you to see how changes in the priorities of criteria affect the overall ranking of alternatives. It provides a dynamic way to visualize the impact of different criteria weights.


4. **Graphs and Charts**: Super Decisions offers various graphical representations, such as bar charts and pie charts, to visualize the priorities and comparisons between alternatives and criteria.


5. **Reports**: You can generate detailed reports that include all the priority calculations and visualizations, which can be exported for further analysis or presentation.


These tools help in effectively communicating the results of the AHP analysis and making informed decisions based on the priorities of alternatives and criteria.
