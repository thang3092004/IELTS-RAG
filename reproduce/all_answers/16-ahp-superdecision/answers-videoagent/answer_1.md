To convert an AHP model into an ANP model in Super Decisions, you need to introduce dependencies and feedback loops among the elements in the decision model. In AHP, the decision model is structured hierarchically, while ANP allows for a network structure where elements can influence each other. The additional judgments required involve assessing the influence of elements on each other, which means you need to make pairwise comparisons not only between criteria and alternatives but also among the elements within clusters and between clusters. This involves creating a supermatrix that captures these interdependencies and calculating the limit matrix to derive the priorities.
