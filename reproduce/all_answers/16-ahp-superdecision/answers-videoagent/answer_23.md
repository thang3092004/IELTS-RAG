When converting an AHP model to an ANP model, new connections are added to reflect the interdependencies among elements. In AHP, the model is hierarchical, with a top-down structure where elements are independent of each other. In contrast, ANP allows for a network structure where elements can influence each other, capturing both feedback and interdependence. This is done by adding connections between elements at the same level or across different levels, allowing for the representation of complex relationships and interactions among criteria and alternatives. These connections reflect the concept of interdependence by acknowledging that the decision elements can have reciprocal influences, which is more representative of real-world decision-making scenarios.
