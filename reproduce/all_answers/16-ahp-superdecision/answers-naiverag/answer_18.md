### Defining and Utilizing Rating Scales in SuperDecisions

When implementing a ratings model in SuperDecisions, defining and utilizing rating scales is a crucial part of the process. This helps in efficiently evaluating multiple alternatives against various criteria. Below is a step-by-step guide on how to accomplish this effectively.

#### Step 1: Create a New Rating Scale

1. **Select the Criterion**: Begin by choosing the criterion for which you want to define a rating scale. For instance, you might select "Prestige" or "Price." 
   
2. **Access the Ratings Tab**: Navigate to the "Ratings" tab within the SuperDecisions interface. This section provides options to manage and create scales for different criteria.

3. **Load or Create a Scale**: You can either load an existing scale or create a new one. For instance, you may want to create a scale ranging from "Excellent" to "Poor." If you're creating one from scratch, you might define points like "Luxury," "Very Good," and "Average," assigning corresponding numerical values for each category.

4. **Define Scale Values**: Enter the actual values associated with each rating. For example, you might rank "Luxury" as 10, "Very Good" as 7, and "Average" as 5. This process allows you to quantify subjective evaluations.

#### Step 2: Inputting and Managing Ratings

1. **Enter Judgments for Alternatives**: Once the scale is defined, proceed to input ratings for each alternative under the selected criteria. For example, when rating a BMW versus an Audi based on "Prestige," you might choose to rate the BMW higher if you perceive it as more prestigious.

2. **Utilize Ratings Models**: When dealing with numerous alternatives, using a ratings model helps manage the complexity efficiently. In such cases, it is often advised to utilize a ratings model to simplify the evaluation process instead of conducting pairwise comparisons, which can be time-consuming and complicated.

3. **Evaluate and Synthesize Results**: With the scales defined and ratings entered, you can synthesize the results using SuperDecisions’ built-in tools. This functionality allows you to see the effective prioritization of alternatives based on the defined scales.

4. **Compare and Adjust**: If necessary, revisit the rating scales and make adjustments based on insights gained during the comparisons. The iterative process ensures better reliability in the evaluation.

#### Step 3: Save and Reuse Scales

1. **Save Scales for Future Use**: After creating a rating scale, save it using the "Save to file" option. This will allow you to reuse the scale in other models or projects as needed.

2. **Manage Multiple Scales**: If dealing with multiple criteria, continue to create, save, and load various scales, ensuring all are appropriately labeled and maintained for easy access during analysis.

#### Conclusion

Utilizing rating scales effectively in SuperDecisions enhances the decision-making process, particularly when evaluating multiple alternatives against several criteria. By systematically defining, implementing, and managing these scales, users can achieve a more structured and reliable evaluation framework that informs better strategic choices. Through practice within the software, users can become adept at creating nuanced and effective rating systems tailored to their specific needs, thereby enhancing the overall utility of the SuperDecisions tool.
