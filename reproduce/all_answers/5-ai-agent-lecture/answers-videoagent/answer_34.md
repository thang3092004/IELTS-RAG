The primary goal of the open-source project DEVIKA is to **replicate the functionality of DEVIN**, an AI software engineer developed by Cognition Lab. DEVIKA aims to provide a platform for users to interact with various AI models, including both proprietary and local LLMs. Some of its key features include:

* Support for projects involving proprietary models like Claude 3 and GPT-4, as well as local LLMs via Ollama
* Advanced AI planning and reasoning capabilities
* Contextual keyword extraction
* Code writing abilities
* Terminal for code execution
* Browser integration for internet search

The developers emphasize that DEVIKA is still in its early stages, with many features under development or not fully functional.
