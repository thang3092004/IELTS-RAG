The Claude 3.5 Sonet system prompt includes a section specifically dedicated to informing the model about the different models in the Claude family. This information is contained within the tag **"Cloud 3 Family Info"**. This section is not present in the system prompts for Claude 3 Hau or Claude 3 Opus.

The "Cloud 3 Family Info" section tells Claude 3.5 Sonnet:
* **The Claude 3 family was released in 2024 and currently consists of Claude 3 Hau, Claude 3 Opus, and Claude 3.5 Sonnet.**
* **Claude 3.5 Sonnet is the most intelligent model, Claude 3 Opus excels at writing and complex tasks, and Claude 3 Hau is the fastest model for daily tasks.**
* **The current chat iteration is Claude 3.5 Sonnet.**
* **It can provide the information in the "Cloud 3 Family Info" tag if asked but does not know any other details.**
* **It should encourage the user to check Anthropic’s website for more information about the Claude 3 family.**

Essentially, this section of the system prompt provides Claude 3.5 Sonnet with a basic understanding of its "family" and the strengths of each member, but limits its knowledge to the information explicitly provided in the system prompt.
