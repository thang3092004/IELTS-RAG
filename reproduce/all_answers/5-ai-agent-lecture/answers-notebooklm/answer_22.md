Using sub-agents and a super agent to analyze financial reports in a PDF format involves a hierarchical approach where smaller, specialized agents (sub-agents) handle specific tasks and a larger, more powerful agent (super agent) orchestrates the entire process. One example of this process is the following:

*   **The user provides a question related to financial data in multiple PDF reports.** For example: *how did Apple's net sales change quarter to quarter in 2023 Financial year and what were the key contributors to the changes?*
*   **A super agent, powered by a large language model (LLM) like Claude 3 Opus, is used to generate prompts for the sub-agents.** The super agent's prompt instructs it to create prompts that will extract specific information from the PDF reports. The super agent should also consider that each sub-agent only has access to one quarterly earning report.
*   **Sub-agents, powered by smaller LLMs like Claude 3 Haiku, are assigned to individual PDF reports.** These sub-agents utilize their vision capabilities to analyze the financial reports. Each sub-agent extracts relevant data from its assigned quarter, focusing on specific aspects like product segment sales and changes from the previous quarter.
*   **The extracted information from each sub-agent is collected and sent back to the super agent.** The collected data is structured using XML tags to separate information from each agent.
*   **The super agent receives the combined insights from all sub-agents and uses them to generate a comprehensive response to the user's question.** This response includes a summary of findings based on the data extracted by sub-agents and may also include visualizations generated using code (e.g., Python code using the Matplotlib library).

This process leverages the strengths of both smaller and larger LLMs. Sub-agents, being smaller and more specialized, can efficiently handle simpler, focused tasks like extracting data from individual reports. The super agent, with its larger capacity and advanced reasoning abilities, can orchestrate the entire process, synthesize information from multiple sub-agents, and generate a comprehensive understanding of the overall financial trends. This approach allows for more efficient and accurate analysis of complex financial data presented in multiple PDF reports.
