Here are some examples of custom GPTs discussed in the sources:

* **Gmar** is a coding assistant that allows users to create websites and other projects with simple sentences. It features hotkeys for coding flows and provides HTML, CSS, and JavaScript code in a single file. The user interface is customizable for programming projects.
* **Nid AI** generates narrated videos for platforms like YouTube, TikTok, and Instagram. It's noted for being distinct from other AI video generators.
* **Consensus** is a research tool that can summarize research papers and provide references. It utilizes information from the Consensus platform and allows users to access the original papers.
* **Scholar GPT**, similar to Consensus, searches a larger database, including Google Scholar and PubMed, for research papers.
* **Canva GPT** can create designs like YouTube thumbnails with multiple AI agents working together. Users can modify the generated designs within Canva.
* **Designer GPT** creates and hosts websites. For example, it can create a homepage for a tech startup and allows users to modify the website.
* **GPTs for AI Agent Frameworks** provide documentation for frameworks like Crew AI and Autogen. These GPTs explain concepts, roles, goals, and execution flows within the frameworks.
* **Logo Creator** generates professional-looking logos and designs based on user prompts. Users can choose from different options and request multiple designs.

The sources also discuss custom GPTs that are replicable using custom prompting and prompt injection. However, the GPTs that are likely to be most useful and successful are those that rely on proprietary data and custom APIs. These GPTs often require users to navigate out of the ChatGPT platform to complete operations. An example of this is **invid AI**, which uses its proprietary APIs and platform.
