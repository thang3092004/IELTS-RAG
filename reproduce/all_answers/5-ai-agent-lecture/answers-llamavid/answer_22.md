The process of using sub-agents and a super agent to analyze financial reports in a PDF format is shown in the video. The first step is to create a new function called `pdf_to_base64_png` which takes a PDF file as input and generates a base64 encoded PNG image. This function is then used by the `pdf_to_text` function to extract the text from the PNG image. The `pdf_to_text` function is then passed to the `parse_capabilities` function which analyzes the capabilities of the model. The capabilities are then used to determine which sub-agents should be used to extract the relevant information. For example, if the report contains information about the company's earnings, the `pdf_to_base64_png` function is used to extract the earnings report. The capabilities are then passed to the `triage_agent` function which determines which agent should be used. The `triage_agent` function then passes the capabilities to the selected agent and the agent generates the final response.
