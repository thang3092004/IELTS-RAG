Anthropic's system prompt design differs from typical approaches in a few key ways. First, it focuses more on the user's background and preferences, using a questionnaire to gather information about their interests and needs. This allows the system to tailor its responses to the user's specific context. Second, it uses a more conversational tone, asking questions like "What would you like to do this weekend?" instead of just providing a list of options. This makes the responses feel more personalized. Finally, the system prompts are designed to be self-contained, so that the user doesn't need to know about other parts of the system. This makes it easy to use the system without understanding the full context. Overall, Anthropic's system prompt design aims to provide a more personalized, conversational, and self-contained user experience.
