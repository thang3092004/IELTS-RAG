The sources detail a number of novel applications of mathematics within art conservation, with some key examples including:

*   **Reconstructing frescoes from fragments**:  Massimo Fonasier pioneered a technique using **circular harmonics** to reconstruct frescoes destroyed during World War II bombing raids. This method allowed for the analysis of the rotation of fresco fragments and the identification of best-fit locations on the original fresco. The reconstructed fresco provides a more complete understanding of the original artwork and its composition.
*   **Automatic thread counting in paintings**: This technique leverages the **Fourier Transform** to analyze the periodicity of canvas threads visible in x-rays. This information can be visualized using a color code representing thread density and orientation, creating a “fingerprint” for the canvas that can be used to identify paintings created from the same roll of canvas (roll mates). This has been used in the attribution and reattribution of paintings by artists like Van Gogh and Velazquez.
*   **Virtual aging and rejuvenation**:  This involves using image processing techniques to simulate the aging process of artwork or reverse it to reveal the original appearance of the work. Examples include:
    *   **Removing cracks:** Cracks in paintings can obscure details, making it difficult to study the underlying artwork. Mathematical algorithms can be used to locate cracks and in-paint the areas to create a crack-free image.
    *   **Color remapping:** The colors of pigments change over time. Analyzing the color shifts of known pigments allows for the creation of a color map that can be used to virtually age or rejuvenate artwork.
    *   **3D modeling and rendering:** This is particularly useful for simulating the appearance of metallic surfaces, such as gold leaf. By modeling the surface in 3D and simulating the reflection of light, a more realistic representation of the original gleaming appearance can be achieved.
*   **Splitting superimposed x-rays**: Paintings created on two sides of a single canvas or panels with paintings on both sides pose a challenge for analysis using x-rays, as both images are superimposed. Ongoing research uses mathematical techniques to learn the correspondence between x-rays and original artwork, aiming to develop an algorithm that can split the superimposed x-ray into two separate images, allowing for the individual analysis of each artwork.

These applications highlight how mathematics can be used to extract hidden information, reconstruct damaged artwork, and create virtual representations that enhance our understanding and appreciation of art.
