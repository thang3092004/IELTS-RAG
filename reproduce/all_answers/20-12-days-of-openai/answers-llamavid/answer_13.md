Sora is a video generation tool that allows users to describe a scene and then specify the actions and camera movements to be performed. The process starts by uploading an image that serves as the starting point. For example, in the image, there are two woolly mammoths standing by a river. The user then sets up a storyboard by adding cards that describe the actions to be performed. The user can specify the actions and camera movements within the cards. For example, the user can specify that the mammoths should walk into the river and catch a fish. Sora then generates the video based on the actions and camera movements specified in the storyboard. The user can then view the final video and edit it to make it match their desired output.
