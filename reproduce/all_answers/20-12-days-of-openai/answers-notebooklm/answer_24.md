Here are some of the improvements made to ChatGPT's search function:

*   **Increased speed**: Search results are now returned faster.
*   **Mobile optimization**: Improvements have been made to the search experience on mobile devices, including:
    *   **Rich visual lists**: Search results for businesses, such as restaurants, now include visual lists with images, open hours, and other relevant information.
    *   **Native Maps Integration**: Search results for businesses can be viewed on a native map within the app. For example, iPhone users will see results in Apple Maps.
*   **Faster web navigation**: When users enter a search query for a specific website, the website links are now displayed more prominently, allowing users to quickly navigate to their destination.
*   **Search integration in advanced voice mode**: Users can now search the web using voice commands in advanced voice mode.

OpenAI is continuously improving the search function based on user feedback. Users are encouraged to provide feedback to further enhance the product.
