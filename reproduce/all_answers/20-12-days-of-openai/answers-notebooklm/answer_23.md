The new ChatGPT integration in Apple's iOS offers several new features that make it easier to use ChatGPT on iPhone and iPad devices. These features include:

*   **Siri Integration:** Siri can now hand off tasks to ChatGPT, streamlining user interaction. When Siri recognizes a complex task that might benefit from ChatGPT, it will ask for confirmation before sending information and then utilize ChatGPT's capabilities. 

*   **Writing Tools Integration:** ChatGPT is integrated with Apple Intelligence writing tools. This allows users not only to refine and summarize documents but also to **compose new documents from scratch using ChatGPT**.

*   **Camera Control Integration:** On iPhone 16, ChatGPT can be invoked through camera control. This **enables visual intelligence, allowing users to point their camera at something and learn more about it using ChatGPT**.

To enable these features, users need to:

*   Enable Apple Intelligence in settings.
*   Enable the ChatGPT extension in Apple Intelligence and Siri settings.
*   Optionally enable confirmation for ChatGPT requests. 

The integration also provides easy access to the ChatGPT app, allowing users to continue and expand on conversations initiated through Siri or writing tools.
