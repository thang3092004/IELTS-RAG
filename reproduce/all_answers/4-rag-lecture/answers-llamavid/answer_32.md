Advanced RAG techniques aim to improve the accuracy and performance of RAG systems by addressing some of the limitations of basic RAG. Some of these techniques are:

1. Late chunking: Late chunking uses a larger context embedding model than basic RAG, which allows it to handle longer input text. However, this requires more computational resources. 

2. Multimodal RAG: Multimodal RAG combines text and image embeddings to generate responses. This can provide more contextual information.

3. GraphRag: GraphRag uses a graph-based approach to summarize the input text. It can handle long input text and multiple entities. However, it requires more data and is more expensive.

4. Re-ranking and re-ranking: Re-ranking and re-ranking techniques can improve the accuracy of the generated responses by re-ranking the retrieved information based on its relevance to the query.

In summary, advanced RAG techniques aim to provide more contextual information and address some of the limitations of basic RAG, such as the need for more computational resources. However, they also require more data and can be more expensive.
