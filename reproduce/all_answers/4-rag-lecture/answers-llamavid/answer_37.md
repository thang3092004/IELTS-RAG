Late chunking, which involves dividing the input text into smaller chunks and then processing each chunk separately, has been shown to improve retrieval system accuracy. The paper argues that late chunking can provide richer and more informative embeddings compared to traditional methods that process the entire text at once. Late chunking also allows the system to handle very long input tokens, which previous models could not support. While late chunking requires more computational resources than traditional methods, it is a technique that can be used to improve retrieval accuracy in retrieval augmented generation.
