The text-based RAG system relies on traditional embedding models to generate responses, while the vision-based system uses vision language models to process the input data and generate responses. 

Text-based RAG:

- Uses traditional embedding models to process the input text and generate responses.
- Relies on standard rack techniques to create context and embeddings.
- Does not leverage vision models for data processing.

Vision-based RAG:

- Uses vision language models to process the input data and generate responses.
- Leverages vision models for data processing and context generation.
- Requires vision models to be trained on large amounts of data.

In summary, the text-based RAG system relies on traditional techniques, while the vision-based system leverages vision models to provide more powerful data processing and context generation. The vision-based system is more expensive and requires more data to be trained.
